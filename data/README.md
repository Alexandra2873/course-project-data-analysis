# Источники данных

Файлы датасетов не размещаются в репозитории из-за объёма и особенностей распространения данных. Для воспроизведения анализа данные необходимо скачать из открытых источников.

## 1. Табличные данные

**Medical Insurance Price Prediction**  
Источник: Kaggle  
URL: https://www.kaggle.com/datasets/harishkumardatalab/medical-insurance-price-prediction

Датасет используется для задачи регрессии страховых расходов на основе характеристик клиента.

## 2. Временные ряды

**SKAB — Skoltech Anomaly Benchmark**  
Источник: GitHub  
URL: https://github.com/waico/SKAB

В работе используется файл `0.csv` из набора данных SKAB. Датасет применяется для анализа многомерных временных рядов и обнаружения аномалий.

## 3. Изображения

**Hard Hat Workers Dataset**  
Источник: Roboflow Universe  
URL: https://universe.roboflow.com/joseph-nelson/hard-hat-workers/dataset/14

Датасет используется для задачи детекции объектов в изображениях. Разметка представлена в формате YOLO.

## 4. Текстовые данные

**Email Spam Classification Dataset**  
Источник: Kaggle  
URL: https://www.kaggle.com/datasets/purusinghvi/email-spam-classification-dataset

Датасет используется для задачи бинарной классификации электронных писем на спам и не спам.
